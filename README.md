### Informações do projeto:
<p>Focado em Backend, desenvolvi em hashtag Python uma aplicação com entrada x processamento x saída de dados. Programei o controle de exceção de erros, não permitindo alertando caso os dados inseridos pelo usuario sejam inválidos.
<br>
Realizei o teste unitário da aplicação, garantindo seu correto funcionamento.
<br><br>
O programa:<br>
- recebe os dados via input do id do funcionário <br>
- calcula o valor a ser pago de seu salário de acordo com o valor de sua hora x o tempo total que ele trabalhou no período.
